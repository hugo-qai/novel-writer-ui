# Novel Writer UI

This is the frontend implementation for the **Novel Writer** project, based on the **Spec-Driven Development (SDD)** methodology.

## Methodology

The UI follows the 7-step SDD workflow:

1.  **Constitution**: Define core principles and values.
2.  **Specify**: Create detailed story specifications.
3.  **Clarify**: AI-driven ambiguity resolution.
4.  **Plan**: Generate technical writing plans.
5.  **Tasks**: Manage prioritized writing tasks.
6.  **Write**: AI-assisted content generation.
7.  **Analyze**: Multi-dimensional quality analysis.

## Tech Stack

-   **React** (Vite)
-   **Tailwind CSS** for styling
-   **React Router** for navigation
-   **Lucide React** for icons

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/hugo-qai/novel-writer-ui.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Project Structure

-   `src/App.tsx`: Main layout and routing configuration.
-   `src/components`: Reusable UI components.
-   `src/pages`: Page components for each stage of the workflow.
